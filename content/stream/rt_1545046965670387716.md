HTML/CSS Tips for &lt;img&gt; 🤳

– Use "onerror" to catch and set a fallback "src" ✨
– Set a "subtle" background-color so users know something is coming 💀
– Use CSS aspect-ratio to avoid layout shifts 🖼️
- Remember your "alt" attribute where necessary 🙏
